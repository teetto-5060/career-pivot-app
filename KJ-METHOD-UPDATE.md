# 🎯 KJ 매핑 방법론 적용 완료!

## ✅ 주요 변경사항

### 1️⃣ Step 2A: 긍정적 문장 프레이밍

**변경 전:**
```
"현장에서 본 고민들을 떠올려보세요"
"고객들이 이런 문제로 힘들어했지..."
예: "스마트폰 뱅킹이 무서워서 은행에 직접 오시는 어르신들"
```

**변경 후:**
```
"해결하고 싶은 문제는 무엇인가요?"
"이런 것들이 해결되면 좋겠는데..."
예: "어르신들이 스마트폰 뱅킹을 편안하게 사용하면 좋겠다"
```

**KJ 매핑 원칙 부합:**
- ✅ 긍정적 프레이밍 (문제 → 해결 희망)
- ✅ 아이디어 발산 촉진
- ✅ 실행 가능성 증가

---

### 2️⃣ Step 2B: 카테고리 제거

**변경 전:**
```
[교육형]
"스마트폰 뱅킹이 무서워서..."
```

**변경 후:**
```
카테고리 라벨 없이
"어르신들이 스마트폰 뱅킹을 편안하게 사용하면 좋겠다"
```

**KJ 매핑 원칙 부합:**
- ✅ 사전 분류 거부 (Bottom-up)
- ✅ 순수한 내용 중심 선택
- ✅ Step 3에서 AI가 자동 그룹핑

---

### 3️⃣ AI 생성 문장: "~하면 좋겠다" 형식

**모든 문장이 긍정형으로 끝남:**
- "어르신들이 스마트폰 뱅킹을 집에서 편안하게 사용하면 좋겠다"
- "대출 서류가 복잡한 청년들이 쉽게 이해하면 좋겠다"
- "보이스피싱을 구별하는 방법을 쉽게 배울 수 있으면 좋겠다"

**효과:**
- 희망적이고 실행 가능한 느낌
- 50-60대에게 부담 없는 표현
- 창의적 사고 촉진

---

### 4️⃣ "사업" → "비즈니스 아이템"

**변경:**
- "사업 정의하기" → **"아이템 정의하기"**
- "맞춤형 사업 기회" → **"맞춤형 비즈니스 아이템"**

**이유:**
- 심리적 부담 감소
- 가볍게 시작하는 느낌
- 린 스타트업 철학 부합

---

### 5️⃣ Step 4: [문제][고객][해결책] 구조

**변경 전:**
```
상담형 (Consultant)
"맞춤형 금융 설계사"
→ 직업명 중심
```

**변경 후:**
```
[어떤 문제를] 노후 자금 관리 불안과 투자 막막함
[누구를 위해] 은퇴 준비 중인 50-60대
[어떻게 해결] 개인 맞춤 재무 설계 상담
```

**린 캔버스 3요소:**
1. 문제 (What) - 빨간 박스
2. 고객 (Who) - 파란 박스
3. 해결책 (How) - 초록 박스

**효과:**
- 실행 가능성 극대화
- 비즈니스 모델 명확화
- 내일 당장 시작 가능한 수준

## 🤖 AI 프롬프트 변경

### generate-problems API
```
역할: KJ 매핑 전문가 + 시니어 창업 컨설턴트
목표: 긍정적이고 확장성 있는 문장 생성
형식: "~하면 좋겠다"
원칙:
- 긍정적 프레이밍
- 확장적 사고
- 구체성
- 사용자 중심
```

### generate-business-types API
```
역할: 린 스타트업 전문가 + KJ 매핑 전문가
구조: [문제][고객][해결책]
목표: 실행 가능한 비즈니스 아이템
출력:
- title: 어떤 문제 (20자 내외)
- description: 누구를 위해 (30자 내외)
- why: 어떻게 해결 (40자 내외)
```

## 🎓 KJ 매핑이란?

**정의:**
- 일본의 문화인류학자 川喜田二郎(카와키타 지로)가 개발
- 영어로는 Affinity Diagram (친화도법)
- 정성적 데이터를 체계화하는 방법론

**핵심 철학:**
1. 데이터가 말하게 하라
2. 사전 분류를 거부하라
3. 긍정적으로 발상하라
4. 구체적으로 만들어라

**적용 사례:**
- 제품 개발
- 문제 해결
- 아이디어 발산
- 사용자 조사
- **창업 아이템 발굴** ← 우리!
